# Regulatory T cells in the tumor microenvironment

## Overview

This is a repository for code to reproduce the figures from "Regulatory T cells modulate macrophages to potentiate immunosuppression in the tumor microenvironment" by Gong et al.

## Downloading data

The data necessary to recreate the figures are available from [our data record on Zenodo](https://zenodo.org/records/21302059). To execute the R markdown files included in this repository, these data should be downloaded and placed into the Data directory.
